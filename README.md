# gMRI
Lightweight implementation of the MRI and gMRI algorithms

## Prerequisites
**gMRI** requires
* **numpy**
* **scipy**
* other standard Python3 modules (**copy**, **warnings**, ...)

## Installing
From the main folder, you can install the package by typing
```
pip install .
```
